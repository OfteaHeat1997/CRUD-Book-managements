# CRUD Project

This is my school project for Fontys University of Applied Sciences.

## React.js & Firebase Realtime Database Tutorial

This project demonstrates how to connect a React.js website to Firebase Realtime Database. It covers the implementation of CRUD (Create, Read, Update, Delete) operations using Firebase with React.

---

## Course Content
1. **Creating React project**
   - Installation of necessary packages
   - Cleanup of React project folder
2. **Creating Firebase project and connecting it to React website**
   - Creating a new Firebase Realtime Database project
   - Configuring React components and files for Firebase connection
3. **Adding Routing**
   - Adding routing to the React website
4. **Writing Data**
   - Pushing new entries to Firebase Realtime Database using input values
5. **Reading Data**
   - Fetching and displaying data in a React component using a map function
   - Understanding Firebase data structure and the `get` method
6. **Update Part - I**
   - Grabbing the identification number for each document/object from Firebase Realtime Database
   - Displaying this ID number on the frontend
7. **Adding Navigation and CSS Styling**
   - Adding navigation between components
   - Applying styling to improve the website's appearance
8. **Update Part - II**
   - Creating the `UpdateWrite` component to update specific objects
   - Using `useEffect` and an update function
9. **Delete Operation**
   - Creating a button and connecting it to a delete function using the `remove` method
